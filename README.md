# DataMarkViz - Dashboard del mercado Mexicano 
  
## Descripcion
DataMarkViz es un dashboard interactivo que permite visualizar el rendimiento real de activos financieros  
desde la perspectiva del inversionista mexicano. Combina precios históricos de acciones (Twelve Data) con  
el tipo de cambio oficial USD/MXN (Banxico) para mostrar el impacto del tipo de cambio en las inversiones.  
  
## Tecnologías  
- **Frontend:** React + Vite  
- **Backend:** SlimPHP 4  
- **Estilos:** Tailwind CSS  
- **Gráficos:** Chart.js  
- **Base de datos:** MySQL  
- **Despliegue:** Vercel (frontend) + Railway (backend)
